MoneyValidator

@Component class MoneyValidator : Validator

Class implementing the validation of Amount of money on ReportedOverlimitTransaction. Property amount must be greater than 0 and less than maxAmount or validator rejects with error code "FIELD_INVALID", and the amount must be less than limit or validator rejects with error code "LIMIT_EXCEEDED". Trieda implementuje Validator rozhranie a validáciu inštancií triedy Money. Premenná amount musí byť viac než 0 a menej než hodnota maxAmount - inak validátor zamietne inštanciu s chybovým kódom "FIELD_INVALID" a amount musí byť menej než hodnota limit - inak validátor zamietne inštanciu s chybovám kódom "LIMIT_EXCEEDED".
