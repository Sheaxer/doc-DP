getNewId

suspend fun getNewId(rep: ReactiveCrudRepository<*, String>, seqName: String): String

Retrieves new value of an id for saving new object in a repository. Updates maximal value of sequence with the given name, checks if an entity with this id already exists in the repository. If it does exist, function finds the actual maximal value of id used to store entities in the repository and updates the sequence.

Parameters

rep - repository where the object will be saved.

seqName - name of the sequence holding the id of last saved object.

Return
value of id that should be used to save object in the given repository.
