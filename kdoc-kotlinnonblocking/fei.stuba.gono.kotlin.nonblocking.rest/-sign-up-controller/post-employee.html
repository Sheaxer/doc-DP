postEmployee

@PostMapping(["application/json"], ["/signup"]) @ResponseStatus(HttpStatus.OK) suspend fun postEmployee(@RequestBody employee: Employee): Employee?

POST method attached to the endpoint /signup. Validates the entity and then saves it if valid. POST metóda pripojená na endpoint /signup. Validuje entitu a ak je korektná, uloží ju.

Parameters

employee - entity representing new employee to be registered as a user of the system. entita reprezentujúca nového zamestnanca ktorý má byť zaregistrovaný ako nový používateľ systému.

Return
entity not containing password if entity was found, null otherwise. entita bez hesla, ak bola nájdená, inak null.
