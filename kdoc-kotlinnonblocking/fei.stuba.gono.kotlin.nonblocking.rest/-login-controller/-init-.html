LoginController(bCryptPasswordEncoder: BCryptPasswordEncoder, employeeService: EmployeeService)

Rest controller that generates JWT that allows access to REST resources upon successfully providing correct username and corresponding password. Rest kontrolér ktorý generuje JWT ktorý umožňuje prístup k REST prostriedkom po úspešnom zadaní používateľského mena a zodpovedajúceho hesla.

Parameters

bCryptPasswordEncoder - Password encoder used to encode the given password to check if is the same as stored password. Enkóder hesiel použitý na zakódovanie doručeného hesla ktoré sa porovná s uloženým heslom.

employeeService - Service used to retrieve and save Employee entities. Služba použitá na získanie a uloženie entít triedy Employee.
