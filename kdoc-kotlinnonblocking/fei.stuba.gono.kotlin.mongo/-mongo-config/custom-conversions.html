customConversions

@Bean fun customConversions(): MongoCustomConversions

Adds custom converters for converting Java Classes that are unable to be serialized / deserialized by MongoDB to classes that are able to be serialized / deserialized. Trieda ktorá pridáva vlastné prevodníky pre MongoDB na konverziu Java tried ktoré nie je možné priamo serializovať v MongoDB.

Return
Instance of class MongoCustomConversions instantiated by list of custom MongoDB converters. Inštancia triedy MongoCustomConvertions vytvorená zoznamom vlastných MongoDB prevodníkov.
