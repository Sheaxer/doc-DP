class AccountValidator

Class implementing validation of ValidAccount annotation. Valid AccountNO must contain either IBAN with optional BIC or local account number. Trieda implementujúca validáciu objektu triedy AccountNO s ValidAccount anotáciou. Platný objekt AccoutNO musí obsahovať buď - IBAN a volitelný BIC - alebo lokálne číslo účtu.
