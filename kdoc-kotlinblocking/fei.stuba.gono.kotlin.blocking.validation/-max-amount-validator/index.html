class MaxAmountValidator

Class implementing the validation for Money object with MaxAmount annotation. Valid amount on a withdrawal must be bigger than zero and less than maximal value. Trieda implementujúca validítor pre object triedy Money s anotáciou MaxAmount. Platný objekt musí mať hodnotu premennej amount vyššiu než 0 a menšiu než maximálna hodnota.
