deleteTransaction

fun deleteTransaction(id: String): Boolean

Deletes the entity with the given id if it exists in the repository and its state is not State.CLOSED. Zmaže entitu so zadaným id ak táto entita existovala a jej stav nie je State.CLOSED.

Parameters

id - id of entity. id entity.

Exceptions

ReportedOverlimitTransactionBadRequestException - in case the entity couldn't be deleted because its state was State.CLOSED. ak entita nemohla byť zmazaná, pretože jej stav bol State.CLOSED
