putTransaction

@PutMapping(["application/json"], ["/{id}"]) @ResponseStatus(HttpStatus.OK) fun putTransaction(@PathVariable id: String, @RequestBody transaction: ReportedOverlimitTransaction): ReportedOverlimitTransaction

PUT method - saves the entity with the given id (if an entity with the given id existed before it will be overwritten). PUT metóda - uloží entitu pod zadaným id (ak už entita so zaaaným id existovala, prepíše ju).

Parameters

id - id that will represent the saved entity. id ktoré bude reprezentovať uloženú entitu.

transaction - entity to be saved. entita ktorá má byť uložená.

Return
saved entity.
