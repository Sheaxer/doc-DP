deleteTransaction

@DeleteMapping(["/{id}"]) @ResponseStatus(HttpStatus.NO_CONTENT) fun deleteTransaction(@PathVariable id: String): Unit

Deletes non-closed ReportedOverlimitTransaction with the request id from database. Zmaže entitu so zadaným id z databázy, ak nemá stav CLOSED.

Parameters

id - id of the entity that should be deleted. id entity, ktorá sa má zmazať.

Exceptions

ReportedOverlimitTransactionNotFoundException - exception thrown if there is no entity with the given id. výnimka vyvolaná, ak entita so zadaným id neexistuje.

ReportedOverlimitTransactionBadRequestException - exception thrown if the entity with given id cannot be deleted because its state is CLOSED. výnimka vyvolaná, ak entita so zadaným id nemôže byť vymazaná, pretože jej stav je CLOSED.
