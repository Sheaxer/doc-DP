WebSecurity

@EnableWebSecurity class WebSecurity : WebSecurityConfigurerAdapter

Class providing security configuration for the endpoints. Sets up authorization using JWTs. Allows unrestricted access to /login endpoint and secures all others with requiring to add Authorization field in HTTP header containing valid JWT with valid credentials of an employee. Trieda zabezpečuje nastavenie bezpečnosti pre endpoint-y. Povolí prístup na /login a /signup endpoint, a zabezpečí všetky ostatné endpoint-y, vyžadovaním pridania Authorization poľa v HTTP hlavičke, ktoré obsahuje platný JWT s údajmi o zamestnancovi
