JWTAuthorizationFilter(authenticationManager: AuthenticationManager, employeeService: EmployeeService)

Class implementing BasicAuthenticationFilter, using JWT. Checks if the Authorization header field of HTTP request is correctly formed e.g. contains "Bearer JWT", verifies the JWT signature and checks if it contains valid employee credentials. Trieda implementujúca rozhranie BasicAuthenticationFilter pomocou JWT. Skontroluje či Authorization hlavička HTTP požiadavky obsahuje "Bearer JWT", kde JWT je korektný JWT a skontroluje, či obsahuje správne údaje o zamestnancovi.
