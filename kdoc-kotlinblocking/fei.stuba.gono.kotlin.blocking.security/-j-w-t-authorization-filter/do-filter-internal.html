doFilterInternal

protected fun doFilterInternal(request: HttpServletRequest, response: HttpServletResponse, chain: FilterChain): Unit

Checks if the HTTP request header has Authorization field that starts with "Bearer " prefix and if so attempts to verify that it contains valid JWT with valid credentials identifying Employee entity with getAuthentication method. Skontroluje, či Authorization pole HTTP hlavičky začína prefixom "Bearer " a ak áno, pokúsi sa verifikovať či zvyšok pola obsahuje správny JWT s korektnými údajmi, zodpovedajúcimi skutočnej entite triedy Employee. pomocou getAuthentication metódy.

Parameters

request - HTTP request. HTTP požiadavka.

response - response

chain - chain
